3 · Animasyon Dosyalarını Ekle
.dotLottie veya .json dosyalarını public/lotties/ altına koyun (meselâ wind-farm.json, solar-track.json).

Recolor gerekiyorsa Lottie Editor’dan Voltura mavisi (#64FFDA) ile yeniden dışa aktarın.

4 · Hero Bileşenine Katman Ekleyin
src/components/AnimatedBackground.jsx

jsx
Kopyala
Düzenle
import Lottie from "lottie-react";
import windFarm from "/public/lotties/wind-farm.json";
import solarTrack from "/public/lotties/solar-track.json";

export default function AnimatedBackground() {
  return (
    <>
      <Lottie animationData={windFarm} loop autoplay
              className=\"absolute inset-0 w-full h-full object-cover opacity-25\" />

      <Lottie animationData={solarTrack} loop autoplay
              className=\"absolute inset-0 w-full h-full object-cover opacity-30 scale-110\" />
    </>
  );
}
Sonra Hero JSX’inde:

jsx
Kopyala
Düzenle
<section className=\"relative ...\">
  <AnimatedBackground />
  {/* H1, H2, CTA */}
</section>
5 · Performans & Erişilebilirlik
Önlem	Kod / Ayar
Lazy Load	<Lottie ... rendererSettings={{ progressiveLoad: true }} />
Reduced Motion	@media (prefers-reduced-motion: reduce) { .lottie { display:none; } }
CLS	Arka plan konteynerine sabit yükseklik (min-h-[800px]) verin.

Lighthouse panelinden LCP & CLS değerlerini kontrol edin (Replit “Webview tools” menüsünde).
